# ubuntu16_rails_config
default configuration ubuntu system from rails

1. install ubuntu 16
2. `set locale: dpkg-reconfigure locales`


3. `sudo sh instal_rbenv+ruby+postgresql+nginx.sh`


> installed:
>  - rbenv
>  - ruby 2.5.0
>  - nodejs
>  - postgresql
>  - sqlite
>  - nginx+passenger
>  - and others....


5. `sudo sh instal_redis.sh`

6. profit!
